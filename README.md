# Proyecto IA – Juego Rullo

Primera entrega del proyecto:  
Se genera un tablero del juego **Rullo** usando JavaScript puro.

## Funcionalidades
- Crea tableros con filas, columnas y dificultad ajustable.
- Muestra sumas de filas y columnas.
- Permite activar/desactivar celdas con clic.

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Cambia los valores de filas o dificultad.
3. Presiona “Nuevo tablero”.
